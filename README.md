### 🛠️ Config
.env
```
botTOKEN=
PORT=3000
OWNER_ID=
MONGODB=
```
### 🖥️ Install
```
npm install
```
### 🚀 Run
```
npm start
```
### 🏷️ Command
- /tkb (tên lớp) (thứ) : xem thời khoá biểu của các lớp khác
- /changeclass : thay đổi tên lớp thông báo
- /changetime : thay đổi thời gian gửi thông báo 
- /help : hiển thị các lệnh
- /start : đăng ký thông báo
